# Supported Sensors

## Introduction

This page is a _un_-complete list of supported sensors by betaflight. To be sure that your sensor is supported check in _/src/main/drivers/drivers/_ _accgyro_, _barometer_, _rangefinder_, _compass_

## Compass

- AK8975
- AK8963
- HMC5883l
- QMC5883l
- HMC5983

## Gyroscope / Accelerometer

- MPU3050
- MPU6050
- MPU6500
- MPU6000
- MPU9250
- BMI160
- BMI270
- ICM20601
- ICM20602
- ICM20649
- ICM20689
- ICM40625
- L3GD20
- LSM6DSO(LSM6DSOX)

## Barometer

- BMP085
- BMP280
- BME280
- BMP388
- QMP6988
- MS5611
- DSP310

## Range Finder

- HCSR04
- Benewake LIDAR TF02
- Benewake LIDAR TFmini
