<TabItem value="notes" label="Notes">

- Manufacturer: iFlight Innovation Technology Ltd. — https://www.iflight-rc.com/
- Product: iFlight BLITZ Whoop F7 AIO (2-6S, 55A) — flight controller with an integrated 4-in-1 ESC (also marketed as "BLITZ F7 55A AIO").
- Integrated ESC: 55A continuous / 65A peak (10 s), BB21 ESC MCU, FD6288 gate driver, ships with Bluejay (Q-H-40) firmware and supports bidirectional DShot (DShot150/300/600). A later V2 board revision ships a BLHeli_S ESC.
- Battery input: 2-6S LiPo / LiHV.
- Power: onboard 5V 2.5A BEC for the FC and peripherals (VTX/camera).
- Onboard current sensor (scale 200, i.e. 1/10 mV/A) — matches the firmware default.
- Board orientation is rotated: config sets `DEFAULT_ALIGN_BOARD_YAW 135`.
