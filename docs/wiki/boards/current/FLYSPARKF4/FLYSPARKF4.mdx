## Other Features

- SD Card Slot: Yes
- Onboard Receiver: No
- Hardware Inverter: Yes
- Bluetooth: Yes
- WiFi: No
- Onboard RGB LED: No

## Input/Output

- USB Connector: USB C
- Motor Outputs: 8x
- UARTs: 4x
- I2C: Yes
- SWD: Yes (SWD - TP5, SWC - TP6)
- SPI: No
- 3.3V Output: Yes
- 4.5V (VBUS) Output: Yes
- 5V Output: 3A
- 9V Output: 3A
- Current Sensor: Yes
- Analog RSSI Input: Yes
- LED Strip Output: Yes
- Buzzer Output: Yes
