## Other Features

- SD Card Slot: Up to 4GB
- Hardware Inverter: Yes
- Power input: 2s - 6s Lipo (10V output will be lower when 2S is used)
- Power output: 3.3V/500mA, 4.5V-5V/2.5A, 10V/3A
- RSSI input: RSSI input solder pad
- I2C: Used for external magnetometer (GPS module), Barometer module
- ESC: 8 pwm pins for motors
- Buzzer: Buz+ (5V) and Buz- pad used for 5V Buzzer

## Input/Output

- USB Connector: USB Type-C
- Motor Outputs: 8x
- UARTs: 6x
- I2C: Yes
- SWD: Yes
- SPI: No
- 3.3V Output: Yes
- 4.5V (VBUS) Output: Yes
- 5V Output: 2.5A
- 10V Output: 3A
- Current Sensor: No
- Analog RSSI Input: Yes
- LED Strip Output: Yes
- Buzzer Output: Yes

<TabItem value="notes" label="Notes">

:::info

**SBUS Inverter**

The SBUS pad is connected to USART2_RX (R2) by an inverter circuit. This inverter circuit can be controlled with CLI commands via Betaflight. If you are going to use R2 as SBUS receiver, please make sure that serialrx_inverted has set as OFF. You can run the following command to turn it ON on or OFF the CLI.

> set serialrx_inverted=OFF

> save

:::

:::danger

When using individual ESCs with integrated BECs, please do not connect the 5V OUT of the ESC to the FC, it will cause the FC/ESC to burn.

:::
