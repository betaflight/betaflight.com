# Orbit F405

## Main Features

- Target: `ORBITF405`
- MCU: STM32F405
- IMU: ICM42688P
- Analog OSD: AT7456
- Barometer: DPS310 or SPL06 (I2C)
- Flash Storage: W25Q128JVPQ (16MB onboard dataflash)
- SD Card Slot: Up to 4GB
- Hardware Inverter: Yes
- Power input: 2s - 6s Lipo (10V output will be lower when 2S is used)
- Power output: 3.3V/500mA, 4.5V-5V/3A, 9V/3A
- RSSI input: RSSI input solder pad
- I2C: Used for external magnetometer (GPS module), Barometer module
- ESC: 8 pwm pins for motors
- Buzzer: Buz+ (5V) and Buz- pad used for 5V Buzzer
- Mounting Holes: M4 30.5x30.5mm
- Dimensions: 36x39mm (W x L)
- Weight: 7.8g

## Input/Output

- USB Connector: USB Type-C
- Motor Outputs: 8x
- UARTs: 6x
- I2C: Yes
- SWD: Yes
- SPI: No
- 3.3V Output: Yes
- 4.5V (VBUS) Output: Yes
- 5V Output: 3A
- 10V Output: 3A
- Current Sensor: No
- Analog RSSI Input: Yes
- LED Strip Output: Yes
- Buzzer Output: Yes

:::info

**SBUS Inverter**

The SBUS pad is connected to USART2_RX (R2) by an inverter circuit. This inverter circuit can be controlled with CLI commands via Betaflight. If you are going to use it as SBUS receiver please make sure that serialrx_inverted is ON. You can run the following command to turn it ON on the CLI.

> set serialrx_inverted=ON

> save

:::

:::danger

When using individual ESCs with integrated BECs, please do not connect the 5V OUT of the ESC to the FC, it will cause the FC/ESC to burn.

:::
