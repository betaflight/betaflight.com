## Other Features

- SD Card Slot: No (uses 16MB onboard flash for Blackbox)
- Onboard Receiver: None (external receiver via UART)
- Hardware Inverter: No
- Bluetooth: No
- WiFi: No
- Onboard RGB LED: None onboard (one WS2812 addressable LED strip output pad provided)

- USB Connector: USB Type-C
- Input Voltage: 4S–8S LiPo
- Motor Outputs: 8x (X8 capable)

- UARTs: 7 defined in target (UART1, 2, 3, 4, 6, 7, 8) — Foxeer lists 6 full RX/TX sets plus an `Rx8` ESC-telemetry pad; the target still defines a TX pin for UART8 (rule 2)

## Pads

Pads are silkscreened `Rn` (UARTn RX) and `Tn` (UARTn TX). The Notes column shows the manufacturer/default recommended function; any UART can be reassigned in Betaflight.

### UARTs

| Name   | Label   | Notes                                                                                    |
| ------ | ------- | ---------------------------------------------------------------------------------------- |
| UART 1 | R1 / T1 | Default serial RC / receiver input (target `SERIALRX_UART`)                              |
| UART 2 | R2 / T2 | Spare / user                                                                             |
| UART 3 | R3 / T3 | Spare / user                                                                             |
| UART 4 | R4 / T4 | Spare / user                                                                             |
| UART 6 | R6 / T6 | Spare / user                                                                             |
| UART 7 | R7 / T7 | Spare / user                                                                             |
| UART 8 | R8 / T8 | ESC telemetry (Foxeer `Rx8`); manufacturer breaks out only the RX pad, but the target defines a TX pin (rule 2) |

<TabItem value="notes" label="Notes">

- Manufacturer: [Foxeer](https://www.foxeer.com/) — [product page](https://www.foxeer.com/foxeer-h7-mpu6000-fc-8s-dual-bec-barometer-x8-g-503).
- This page documents the **full-size** Foxeer H7 MPU6000 (37 x 37 mm, 30.5 x 30.5 mm mounting). Foxeer also sells a [**Mini** version](https://www.foxeer.com/foxeer-h7-mini-mpu6000-fc-8s-dual-bec-barometer-g-504) (approx. 30 x 31 mm, 20 x 20 mm mounting, ~5.6 g) under the same product name — verify which variant you have before relying on the physical dimensions above.
- The gyro ships as an MPU6000; the `FOXEERH743` target also builds in MPU6500 and ICM42688P drivers for auto-detection across hardware revisions.
- The OSD chip is an inferred MAX7456-compatible clone (commonly an AT7456E): config only defines `USE_MAX7456` and Foxeer's spec lists it simply as "BF OSD". The exact part is not confirmed from a marking.
- Dual BEC: 5V/2A for the flight controller and peripherals, 10V/2A for the video transmitter, plus a 3.3V/0.5A rail.
